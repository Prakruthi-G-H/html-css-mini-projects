## Rotating Text Animation 🎨✨

A simple and stylish rotating text animation built using **HTML5** and **CSS3**.

This project demonstrates how to create a smooth **vertical rotating text effect** using CSS keyframe animations without any JavaScript.

## 🔧 Technologies Used

- **HTML5** – Structure of the webpage
- **CSS3** – Styling, Flexbox layout, responsiveness, and keyframe animations  


## 📁 Folder Structure
- `index.html` → Main HTML file
- `RotatingText.css` → Main stylesheet 
- `README.md` → Project documentation


## 📌 Features 

- 🔄 **Smooth Vertical Text Rotation** using CSS keyframe animations
- 🎨 **Built with Pure HTML & CSS** – No JavaScript required
- 📱 **Responsive Design** – Automatically adjusts for desktop, tablet, and mobile devices
- 📦 **Lightweight & Beginner-Friendly** – Easy to understand and customize


## 🚀 How to Run
  
1. Download or clone the repository:  
     ```bash
   git clone https://github.com/Prakruthi-G-H/html-css-mini-projects.git
   ```
 2. Navigate to the project folder:

   ```bash
   cd rotating-text
   ```
 3. Ensure both files are present in the folder:
    - index.html
    - RotatingText.css
 4. Open index.html in any modern web browser (e.g., Chrome, Firefox, Edge)
